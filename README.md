# Vue with typed translates

This project covers the fundamental principles of writing recursive types.
This project is non-usable and demonstrates only how to set up some tricky TypeScript code in your own projects.

If you want to use this code, please leave references to this repository and leave copyright info.

If you want to try how it works, follow the next instructions:

- copy repository, using `https://github.com/RostyslavSharuiev/translates-app.git`
- open project in your IDE
- run `npm i` in command line
- run `npm run dev` and enjoy
